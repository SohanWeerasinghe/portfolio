Task Management System

The system uses Role-Based Access Control (RBAC) with Admin and User roles. Admins can manage tasks, users, and perform CRUD operations, while Users can manage their own tasks. Tasks can be created, updated, deleted, and filtered by user ID, with details like title, description, and due date. Admins also handle user management, including adding, updating, and deleting accounts and roles. The system uses JWT for secure login and role-based API protection. Input is sanitized using DOMPurify (frontend) and StringEscapeUtils (backend), and CSRF tokens protect sensitive actions. Passwords are securely hashed with BCrypt, and security headers are added to protect against unauthorized access.
